JavaScript Interactive Features Showcase
This project demonstrates a collection of advanced, user-friendly interactive features built with vanilla JavaScript. Designed to elevate user experience and accessibility, it includes the following functionalities:

Dark Mode Toggle: Effortlessly switch between light and dark themes for optimal visual comfort.
Responsive Image Slider/Carousel: Smooth auto-play with manual navigation controls to showcase images seamlessly.
Modal Popups: Dynamic popups with open and close capabilities, triggered via buttons and dismissible by clicking outside.
Real-Time Form Validation: Instant feedback on user input for name, email, and phone fields, ensuring data accuracy.
Smooth Scrolling Navigation: Elegant transition effects for anchor links, enhancing site navigation.
Animated Counters: Visually appealing counters that animate upon entering the viewport, perfect for displaying statistics.

Built with a focus on clean, efficient, and accessible code, this project serves as a comprehensive example of implementing interactive web components using only native JavaScript. Ideal for developers seeking to enhance their front-end skillset or incorporate these features into their projects.
